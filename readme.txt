# PBUrTBots plugin
# Plugin for B3 (https://github.com/BigBrotherBot/)
# www.ptitbigorneau.fr

PBUrTBotsplugin (v1.1) for B3

Requirements
------------

* BigBortherBot(3) >= version 1.10

Installation
------------

1. Copy the 'pburtbots' folder into 'b3/extplugins' and 'pburtbots.ini' file into '/b3/extplugins/conf'.

2. Open your B3.ini or b3.xml file (default in b3/conf) and add the next line in the [plugins] section of the file:
    for b3.xml
        <plugin name="pburtbots" config="@b3/extplugins/conf/pburtbots.ini"/>
    for b3.ini
        pburtbots: @b3/extplugins/conf/pburtbots.ini

3.  Copy q3ut4/listmapsbots.txt into your q3tu4 folder or create a file with maps compatible with bots
